# goutil
是go 的工具集,也叫小函数

## 目录要求
- 一个类别一个文件夹,文件夹里必含一个测试文件
- 如array

```
 |- array
        |- array.go      //核心文件
        |- test.go //测试文件
```

## 函数分类列表

| 功能 | 包名 |  备注 |
| :--- | :--- | :--- |
| 数值转换 | [convertor](convertor) | 操作数字等 |
| 时间 | [datetime](datetime) |  操作时间等|
